Kişisel portfolyo sitesi. HTML ve CSS ile yapıldı.

## Canlı Demo
Yakında eklenecek.

## Kullanılan Teknolojiler
- HTML5
- CSS3 (Flexbox, Box Model)
- Google Fonts (Inter)
- Git & GitHub

## Proje Yapısı
- index.html — Sayfa içeriği
- style.css — Tasarım ve stiller
- README.md — Proje açıklaması

## Özellikler
- Semantic HTML yapısı
- Dark mode tasarım
- Sticky header
- Hakkımda, Projeler, İletişim bölümleri
- İletişim formu
- Hover animasyonları

## Kurulum
Projeyi lokalde çalıştırmak için:

1. Repoyu klonla
   git clone https://github.com/EmreGokcek/personal-landing-page.git

2. Klasöre gir
   cd personal-landing-page

3. index.html dosyasını tarayıcıda aç
